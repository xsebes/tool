0 Predpoklady
  1. Všetci študenti v súbore študenti.csv sú riadne zapísaní na predmet.
  2. Kontrolované aplikáciou:
    1. Email je vyplnený.
    2. Email sa v zozname neopakuje.
    3. Téma 1, Téma 2 a Téma 3 sú celé kladné čísla.
    4. Téma 1, Téma 2 a Téma 3 sú v každom riadku navzájom odlišné.
    5. Pri rovnakej kapacite sú ID tém v rozsahu 1 až n.
    6. Pri kapacite z témy.csv existujú vybrané ID v zozname tém.
    7. Kapacity v témy.csv sú celé kladné čísla.
  3. Téma n je celé číslo ID témy, o ktorú má študent záujem. Téma 1 je ID témy s najvyššou prioritou.
  4. Takýto formát vie zabezpečiť napríklad Google Forms dotazník s validáciou odpovedí. Výstupom je CSV dokument so stĺpcami Email, Meno a priezvisko, Téma 1, Téma 2, Téma 3.
